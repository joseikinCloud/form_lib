# 1.4.0 リリースノート
## Added
  役員情報を会社情報から参照する機能
  下記の項目を会社情報から参照する機能
  + 給与支払い日
  + 助成金担当者役職
  + ゆうちょ銀行利用フラグ
  + 金融機関名
  + 金融機関コード
  + 支店名
  + 支店コード
  + 口座番号
  + ゆうちょ銀行の記号
  + ゆうちょ銀行の番号
  + 口座種類
  + 口座名義漢字
  + 口座名義カナ
  + 資本金（万単位）
  「マス目で区切る」にチェックを入れた場合、文字を等間隔で配置する機能
  様式情報に cover がついているか判定する機能
  支給額と対象労働者をコンソールに表示する機能
  プレースホルダーに値を格納する機能
## Changed
  会社情報が空の場合、オブジェクトの値を空で上書きしない（ビル名を除く）
  別会社に複製した場合、会社情報が空の項目を空で上書きする
  オブジェクト名の末尾に_0を付けられない問題の解消
